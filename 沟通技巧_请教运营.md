# 请教运营的沟通技巧

## 核心原则

**让人一句话就能答，别让人思考太多。**

## 怎么问

**三段式：请教一下 → 选择题 → 为什么困惑**

- **问小不问大** — "满减第一档卡几折？"而不是"满减策略怎么做？"
- **给选项** — "先冲单量还是先保利润？"而不是"新店应该怎么做？"
- **说清楚为什么困惑** — 让对方知道你不是瞎问，是真的想不通
- **一次一个** — 不要连环追问

示范：

```
想请教你个事——新店前期是先冲单量还是先保利润？
我看到两种说法，一个说先活后赚，一个说利润才重要，不知道该听哪个。
```

```
请教下，判断一家店产品好不好，你一般看复购率还是看差评？
我想用复购率，但新店数据太少好像不准。
```

```
问你个小问题，满减第一档一般卡客单价的几折？五折还是六折？
我看竞对有高有低，不确定什么算合理。
```

```
想问下，好评返现新店能用吗？
我学到的说法是伤品牌不该用，但又看到新店推荐用，有点矛盾。
```

## 别人回答后

1. **先说你的理解** — "明白了，就是说先把量跑起来才有数据"
2. **看对方是否愿意继续** — 愿意聊→追一个小问题深入；不想聊→收
3. **追问也要小** — "那一般跑多久算稳了？一个月？"

## 什么时候问

- 刚完成一个任务、结果正常、运营不忙的时候
- 自然衔接上下文 — 刚做完竞对分析就聊竞对相关的，不要突然跳话题
- 同一天最多一次
- 运营不回就算了，不追

## 什么时候不问

- 运营正在忙、刚安排了任务
- 刚报完一堆问题
- 运营语气急

## 记录

运营回答了有价值的 → `python3 learn.py knowledge "内容"`
运营纠正了你的理解 → `python3 learn.py feedback "内容"`
