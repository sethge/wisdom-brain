## 一、已确认认知

### 1. 漏斗与新老客

**L-1** 漏斗数据第一件事拆新老客。综合指标是假象——进店率9.3%拆开是新客6.06%+老客27.59%。两个方向完全不同。
- 来源：酸菜猪脚饭+侬晓得（20260420）

**L-2** 新客链路：曝光 → 进店率 → 下单率 → 首单
老客链路：存量 × 复购率 × 复购频次
两条链路的问题诊断和解法完全不同，不能混用。

**L-3** 满减的本质是升级引导，不是进店吸引。2024.10美团改版后列表页不展示满减标签。
满减作用：顾客犹豫要不要点更贵组合时降低升级心理门槛。
满减要设在顾客可能犹豫的价格节点上——38-1没意义（¥1不改变任何人的感受）。

**L-3-1** 满减第一档设计公式（两步校验）：

步骤一：从实付客单倒推目标
1. 目标实付 = 当前实付客单 × 1.15（+15%为基准）
2. 满减力度约10%，门槛 = 目标实付 ÷ (1 - 力度) → 取整
3. 示例：实付¥30 → 目标¥34 → 门槛≈¥38

步骤二：用"1.3倍上限"校验门槛是否合理
- 用户只愿意在当前价格上限基础上提升30%：门槛上限 = 当前价格带上限 × 1.3
- 若步骤一算出的门槛超过这个上限 → 门槛要往下调
- 示例：盖饭¥26.8 → ×1.3 = ¥34.8 → 第一档门槛不能超过¥34 → 满32减3 或 满34减4

**核心原则：满减第一档只能更低不能过高，宁可让利多一点也要触发。**
- 来源：湘湖知味设计讨论（20260420）

**L-4** 品类名太窄会卡新客进店。侬晓得"排骨年糕"进店率6.71%低于均值7.50%。
店名/logo要能覆盖更大的需求面（实际还有菜饭/拌面/馄饨）。
- 来源：侬晓得（20260420）

### 2. 产品与菜单设计

**P-1** 顾客需求的第一原理是生活场景，不是产品特征。
"想吃丰富解馋的午餐" = 对的出发点
"想要有肉有蛋有卤味" = 本末倒置（从产品出发）

**P-2** 菜单设计要看品类特征，不能套模板。
- 猪脚饭（重主体品类）→ 适合"主体+配角+小菜"套菜逻辑
- 鸭血粉丝（一碗里有很多东西）→ 适合全家福逻辑
不同品类的组合方式不同。

**P-3** 口味DIY是一个独立的设计维度，和全家福（丰富度）不冲突。
全家福解决"吃什么"，DIY解决"怎么吃"（卤汁/辣酱/加料自选）。
类似奶茶的糖度+加料，可以是差异化记忆点。
- 来源：酸菜猪脚饭（20260420）

**P-4** 口味和丰富度不能割裂。套菜应该是整体（"酸菜猪肘+卤蛋+小菜"），不是主菜加一堆散装加购品。

**P-5** 套餐设计要用订单关联数据做科学搭配。不是拍脑袋组合。后台有单品数据，可以挖真实高频搭配。
商品助手的套餐推荐可作灵感但不照搬。
- 来源：侬晓得（20260420）

**P-6** 品多价杂的品类（陕西小吃/快餐），套餐的核心价值是简化决策。
参考肯德基自选套餐：主食选1+小食选1+饮料选1，打包定价。顾客不用算价格，选一选就下单。
件均高（3.15）说明搭配消费已经在发生，套餐只是把顾客已有的行为产品化。
- 来源：陕西风味（20260421）

### 3. 数据判断标准

**D-1** 基准线（非绝对，仅参考）：
- 复购率：20%+ 才算高
- 进店率：10%+ 才及格；新客进店率6%左右偏低
- 评分：4.6+ 才正常；4.5 偏低
- 老客下单转化率：50%+ 健康

**D-2** 神抢手一口价占比基准：
- 10%左右：正常
- 20%以内：冲量期/竞争激烈期可接受
- 超过20%：有问题，利润被持续压缩

**D-3** 减配送费不是商家活动资源，是平台强制/配合项，不计入商家活动力度分析。诊断时不讨论。

**D-4** 福利品（收藏有礼等）的加购率不能当需求信号。几乎免费的东西加购高是活动机制驱动，不是真实需求。
验证真实需求：看正常定价下的加购率或套餐占比。
- 来源：酸菜猪脚饭（20260420）

**D-5** "A卖得好B卖不好"不等于"顾客不要B"。可能只是A更优吸走了流量。
自选双拼16 vs 固定双拼2 → 结论是"自选更有效"，不是"顾客不接受被安排"。

**D-6** 下单率高不等于产品力强。要看到手价格——25.8%下单率+¥21.63到手价可能是价格驱动。

**D-7** 复购率/转化率基准要同时看商圈均值和前10%两个基准，不能只看均值证明自己好。
低于前10%≠下滑，是增长潜力；要区分"有问题"和"有空间"。

**D-8** 到手客单价不能脱离品类成本结构判断利润。肉夹馍到手¥18.49，看单价低，但食材成本低、月营收¥43k毛利约¥30k。要看月毛利总额，不能只看单均到手。
- 来源：陕西风味（20260421）

**D-9** 活动投入看到手率（营收/优惠前总额），不看活动力度绝对值。
陕西风味到手率45.9%（营收¥43k/优惠前¥93k），直观反映"一半被活动吃掉"。比"活动力度37.95%"更有判断力。
- 来源：陕西风味（20260421）

**D-10** 拼好饭的本质是低价获取新客+推广新品的手段，不是常规销售渠道。
判断要不要做拼好饭，不是看"品类适不适合"，而是看：①当前单量够不够 ②新客比例是否需要提升。
拼好饭客群价格敏感，到手低，老板觉得单量够+不缺新客 → 不需要做。
- 来源：陕西风味（20260421）

### 4. 竞对分析

**C-1** 竞对分析的价格带视角：¥20-35一人食午餐客群在茶餐厅/卤煮/饺子/猪脚饭之间换着吃，全是竞对。不要只看同品类。

**C-2** 竞对竞争性质要看数据，不套结论。
- 流向数据显示口味相近品类（猪脚/肘子/卤煮）→ 口味偏好竞争
- 这不是通用规则，每家店看自己的竞对流向判断

**C-3** 客单价是结果不是目标。核心是满足顾客需求，他点了的结果就是客单。不要说"提客单价"。

---

### 5. 评价与口碑

**R-1** 评分4.6+ 才正常。4.5偏低，会影响全链路。

**R-2** 差评分析看比例不看绝对数。单量大时差评绝对数自然高，要看差评率（差评数÷总评价数）才有意义。

## 二、模糊待确认认知

> 这些是有推测但没有验证的认知。遇到相关案例时主动提问确认。
> 格式：问题 | 当前推测 | 待确认点 | 状态

### 认知类模糊项

**F-1** 新店期追单量 vs 保利润的判断条件
- 已知：竞争激烈品类（陕西面食案例）需要先冲量占位
- 待确认：什么品类/什么条件下该追量，什么时候切利润？判断标准是什么？
- 状态：⏳

**F-2** "匹配"的通用操作流程
- 已知：17品类有手册
- 待确认：遇到第18个新品类，第一步做什么？怎么判断匹配成功？
- 状态：⏳

**F-3** 菜品结构"四角色" vs "君臣佐使"的关系
- 引流品/主力品/利润品/凑单品 vs 君/臣/佐/使
- 待确认：同一套不同叫法？还是两套独立逻辑？映射关系是什么？
- 状态：⏳

**F-4** 四率优化和阶段策略如何对齐
- 四率：曝光→进店→下单→复购；阶段：新店→增长→稳定→沉底
- 待确认：每个阶段重点优化哪个率？需要一张交叉表
- 状态：⏳

**F-5** 满减该不该递进的判断条件
- 待确认：什么品类/情况下满减要递进？什么时候不递进？判断标准是什么？
- 状态：⏳

**F-6** 老客转化率健康标准是否因品类而异
- 已知通用基准：老客下单50%+；7日复购15%，30日复购20%+
- 待确认：快餐/茶饮/烧烤的基准线分别是多少？
- 状态：⏳

**F-7** 活动力度指标的计算口径和评判标准
- 已知：美团后台显示"活动力度34.48%"，但这个数字的计算方式不明确——是营销支出/营业收入？还是折扣金额/GMV？包含哪些活动类型（满减/神抢手/配送费/券）？
- 待确认：正确的口径是什么？多少算高/合理/低？如何判断活动投入是否有效？
- 来源：湘湖知味诊断（20260420）
- 状态：⏳

## 三、模糊认知请求逻辑

> 在做诊断或日常沟通中，遇到以下场景时主动判断是否能帮助确认模糊项。

### 触发规则

```
每次分析一家店的数据时，扫描以上模糊待确认项（F-1 到 F-6）：
- 当前案例是否能帮助验证某个模糊项？
  - 是 → 提出问题："这家店的情况可以帮我验证一个我不确定的认知：[描述F-X]，你觉得这里的判断逻辑是什么？"
  - 否 → 继续，不打断
```

### 触发时机
1. 诊断结束后，TODO确认前
2. 日常讨论中遇到相关品类/场景
3. Seth主动提到某个运营决策时

### 请求语气原则
- 带着自己的推测去请教，不空问
- 格式："我的理解是[X]，但[Y]点我不确定——这里的判断逻辑是什么？"
