# 外卖运营认知框架 v1.1

> 基于208个外卖运营课程文字稿提炼 + Seth实战修正。
> 核心原则：运营认知不是规则本身，而是**判断什么条件下用什么规则的能力**。

---

## A. 运营认知分级

### 第零层：运营的本质（道）

**运营的本职工作是：让好吃被更多人吃到。**

这句话拆开就是完整的运营逻辑链：

1. **前提是"好吃"。** 东西不好吃，复购低，拉新只会成本越来越高——商圈存量有限，拉过来吃一次不回头，还带来差评，恶性循环。复购率高，客户吃了就会复购，单子滚雪球，良性循环。所以**产品力是一切的地基**。

2. **运营的核心工作是"匹配"。** 在有限资源下，找到商家能出餐的产品，以高效率的方式匹配到对应的客户群。
   - 找到**商家的定位核心**（他擅长什么、能稳定出什么）
   - 找到**客户的外卖点餐场景**（一人食？家庭餐？商务餐？夜宵？）
   - **做匹配**：把商家的能力翻译成客户需要的场景
   - 例：大饭店的大盘菜→外卖是一人食场景→出对应的一人套餐→定位"干净放心的商务套餐"→强调老店信得过属性→提高转化和复购

3. **"被更多人吃到"是最后一步。** 先把匹配做好（转化率和复购率），以此为基础再增加曝光，让更多人看到→尝试→复购。顺序不能反——基础没打好就拉曝光，等于往漏桶里灌水。

**一句话：先有好产品，再做好匹配，最后放大曝光。这就是运营。**

---

### 第一层：基础认知（生存线）

这是做外卖的"入场券"，不具备这些认知的店铺大概率活不过三个月。

1. **外卖是一门独立的生意，不是堂食的附属品。** 它有独立的成本结构、独立的菜单、独立的人员配备、独立的厨房动线。把堂食菜单照搬到外卖上，是最常见的致命错误。
2. **外卖的本质是效率，不是手艺。** 堂食讲选址和服务体验，外卖讲流量获取和转化效率。外卖更接近电商，不是传统餐饮。
3. **外卖四率模型是基本功。** 曝光率→进店率→下单率→复购率，每一步都有漏斗，每一步都有可优化的因子。不理解四率的人无法诊断问题。
4. **成本决定生死。** 平台抽佣18-23%，加上食材、房租、人工、包装、配送，毛利率被压缩到17-22%。成本管控不是"省钱"，是"活下来"的前提。
5. **菜单决定一切的起点。** 菜品结构、定价逻辑、分类排序、引流品设计——这些在店铺上线前就必须想清楚，上线后再改代价极大。

### 第二层：进阶认知（增长线）

具备这些认知的运营者能把店铺从"活着"推向"增长"。

6. **转化率是权重之王。** 平台的本质逻辑：谁能把流量高效转化为订单，平台就给谁更多流量。转化率>客单价>店铺评分>基础操作，这是权重因子的优先级。
7. **满减不是让利，是流量工具。** 满减的核心作用是拉高进店率（在列表页展示时的视觉竞争力），其次才是促进下单。满减设计要对标竞对，综合满减权重分值要高于竞对。**满减是否递进要分品类看**，一般品类应递进，但不是绝对标准。
8. **数据驱动的闭环思维。** 数据→运营调整→产品调整→市场反馈→新数据。这个循环是外卖运营的核心方法论，不是拍脑袋做决策。
9. **复购是根本，拉新是竞争策略。** 外卖是区域存量市场（周边3-5km固定人口）。复购合格情况下，拉新是最主要竞争手段；复购不合格，先修复购。复购是根基，拉新是攻势，顺序不能反。
10. **推广是放大器，不是救命药。** 沉底店开推广没有用——推广放大的是转化能力，如果基本面（菜单、活动、评分）没做好，推广只是烧钱。

### 第三层：高阶认知（利润线与壁垒线）

这些认知区分"运营高手"和"普通运营"。

11. **定价法取决于竞争格局。** 品牌力不强/竞争激烈的普通品→从成本出发定价；只有你有别人没有的差异化产品→从消费力出发定价。两种都要不断测试，找到自己的合适区间价格。
12. **可以有爆品，不能只有爆品。** 爆品本身是好事（引流+口碑），但单品依赖度>70%很危险——竞对跟进压价→利润崩塌。正确做法：爆品引流，利润分散到所有商品。关键看**爆品占比**，不是有没有爆品。
13. **用户分层运营。** 高消费力用户追求品质、有忠诚度；低消费力用户追求价格、无忠诚度。对高价值用户提供超预期体验，对价格敏感用户保障食品安全即可（低品质≠劣质）。
14. **先验性价值与后验性价值。** 下单前的"看起来值不值"（图片、装修、评价、价格）决定购买率；收到后的"实际体验好不好"（口味、包装、配送、服务）决定复购率。两者必须匹配，否则一锤子买卖。
15. **外卖赚钱的本质路径只有三条：** ①保本冲量+后期提价（慢且风险高，提价必流失）；②亏本冲量+招加盟割韭菜（品牌公司套路）；③成本优势+数据优势+存量复购（唯一可持续路径）。

## B. 核心价值观

### 应该信的（正确认知）

| 编号 | 核心信条 | 说明 |
|------|---------|------|
| V1 | **效率为王** | 外卖的一切竞争最终都是效率竞争：出餐效率、流量转化效率、资金使用效率。 |
| V2 | **数据说话** | 不靠感觉做决策。进店率不行调前端（满减/配送费/logo），下单率不行调后端（菜单/定价/图片）。数据告诉你该做什么。 |
| V3 | **成本是护城河** | 同样的品质，谁的成本更低，谁就有更大的定价空间和利润空间。成本优势是竞对最难复制的壁垒。 |
| V4 | **复购是命脉** | 拉新成本远高于维护老客成本。一家健康的外卖店，老客占比>60%，复购率是平台给权重的核心因子之一。 |
| V5 | **体验是整体的** | 外卖卖的不是菜品，是菜品+包装+配送+服务的整体体验。任何一个环节拉垮，整体体验就崩。 |
| V6 | **先活后赚** | 新店第一目标是单量（证明转化能力），第二目标才是利润（提升客单价）。顺序不能反。 |
| V7 | **菜单是战略** | 菜单不是"把能做的菜列上去"，是精心设计的转化路径：引流品吸引进店→主力品承接需求→利润品贡献毛利→凑单品提升客单。 |

### 不应该信的（常见误区）

| 编号 | 错误认知 | 真相 |
|------|---------|------|
| X1 | "外卖是伪需求/不赚钱" | 不是外卖不赚钱，是你的菜单和活动设计不合理。全国400万+商家、日均3000万+订单是真实需求。 |
| X2 | "单量越多越好" | 订单量不重要，实收额才重要。300单亏钱不如100单赚钱。 |
| X3 | "堂食菜单可以直接用" | 外卖需要专属菜单，专属定价，专属产品组合。就像淘宝有"电商专供"，外卖要有"外卖专供"。 |
| X4 | "做个爆款就能起飞" | 可以有爆品，不能只有爆品。单品依赖度>70%时风险极高。爆品引流+利润分散才是正确结构。 |
| X5 | "推广能救沉底店" | 推广放大的是基本面。基本面差的店，推广只是加速烧钱。 |
| X6 | "好评返现是好策略" | 用户会认为"你利润很高才给我返现"→对产品价值产生怀疑→长期伤害品牌。 |
| X7 | "只用一种定价法" | 普通竞争品从成本出发，差异化独有品从消费力出发。不是非A即B，是看竞争格局选策略，然后不断测试找合适区间。 |

## C. 关键认知清单（30条）

### 【底层逻辑】

1. **外卖=餐饮+互联网，本质偏电商。** 讲究流量、转化、复购，不是选址和装修。
2. **外卖的成本结构和堂食完全不同。** 堂食的大头是房租和人工，外卖的大头是平台抽佣和配送。必须重构成本模型。
3. **互联网让价格透明化，一切行业趋向平均利润。** 不要想超额利润，要想"在平均利润下如何通过成本优势活得更好"。
4. **外卖是区域存量市场。** 你的用户池有上限（周边3-5km固定人口），不像电商可以全国卖。所以复购比拉新重要十倍。
5. **平台的排名逻辑=千时千面+千地千面+千人千面+推广位。** 自然排名由销售额（进店率×下单率×客单价）决定，不是由你给平台多少钱决定。

### 【菜单与定价】

6. **中心价格带必须清晰。** 让用户一眼看出"在你家吃一顿大概多少钱"。价格带模糊→用户犹豫→流失。
7. **菜品结构四角色：引流品、主力品、利润品、凑单品。** 引流品不赚钱但拉人进店，利润品藏在菜单中部，凑单品提客单价。**引流品效果好值得肯定，但同时要关注到手价过低带来的价格锚定风险和长期亏损隐患。**
8. **外卖菜单要"做减法"，但分品类看。** 一般品类一级分类6-8个。但加SKU品类（麻辣烫/串串/麻辣香锅等DIY品类）天然需要大量SKU，不能用统一数字卡。关键是**分类是否清晰**，不是SKU多不多。
9. **菜品命名有四种方法：口味命名、食材命名、地域命名、功能命名。** 好的命名能让用户"在名字里就完成了想象中的品尝"。
10. **定价要和满减联动设计。** 单品定价法：主食价格≥第一档满减门槛（单点即满减）。平均定价法：需两个商品凑满第一档。不同品类适用不同策略。

### 【满减与活动】

11. **满减影响下单率，不是进店率。** 满减在列表页有展示，但真正影响的是进店后的下单决策——"优惠力度够不够"决定买不买、买多少。满减权重分值=各档满减率的平均值。（2026-04-10修正：此前误认为满减主要拉进店率）
12. **满减四种设计模型：** ①门槛相等+金额略大于竞对 ②金额相等+门槛略低于竞对 ③门槛和金额都低于竞对 ④混合自由设计。核心是综合满减率要高于竞对。
13. **满减陷阱设计。** 每档只差5元门槛但多减2-3元→用户不知不觉拉高客单价。麻辣烫/麻辣香锅等DIY品类特别适用。
14. **活动不是越多越好，是越"有节奏"越好。** 新店期大力度活动引流→稳定期调整为精准活动→成熟期控制活动成本保利润。

### 【流量与转化】

15. **曝光来源至少9个入口。** 商家列表、品类频道、搜索、订单详情、好友推荐、优惠专区、品质专区、banner广告、为你优选。每个入口都有争夺价值。
16. **进店率的五个可控因子：配送费、起送价、评分、店名、logo。** 距离和配送时间不可控，这五个可以优化。（2026-04-10修正：满减影响下单率不是进店率，从进店因子中移除）
17. **下单率=尝试成本+先验性价值。** 尝试成本低（价格OK、选择简单）+先验性价值高（图片好、销量高、评价好）→购买率高。
18. **转化率是平台给权重的第一因子。** 你给平台1000曝光能转化50单，隔壁只能转化10单——平台会把更多曝光给你。这就是为什么"先做好基本面再开推广"。

### 【推广策略】

19. **点金推广的核心作用是"让人进店"，不是"让人看到"。** 曝光不扣费，点击才扣费。所以推广前必须确保进店后的转化率够高，否则花了点击费但没有订单。
20. **自助推广优于平台自动推广。** 系统推荐出价通常虚高，实际出价可以低很多（系统推荐3.5元，实际0.8-0.9元即可）。高曝光模式花钱快但不精准。
21. **分时推广策略。** 非主营时段竞争少、出价低、但排名依然有效。中餐店在下午2-4点开推广是"差异化竞价"的好时机。

### 【沉底店复活】

22. **沉底店的病因诊断四问：价格错误？定位错误？品类选择错误？路线规划错误？** 不同病因对应不同解法，不能一刀切。
23. **沉底店复活的正确顺序：先修基本面→再用推广+补单拉曝光→等转化数据跑起来→再做时收提升。** 单量是基础，时收是升级，顺序不能反。
24. **沉底店最常见的错误路径：直接开新店特权→没有数据支撑→特权结束后断崖下跌→盲目开推广→烧钱无效→放弃。** 正确做法：先铺好转化基础，再有节奏地打开流量入口。

### 【评价与体验】

25. **评分4.7以上为健康，4.5以下为危险。** 评分影响进店率（列表页展示）和权重（排名因子）。差评管理不是"回复"，是"追溯问题根源并解决"。
26. **顾客体验地图四个触点：下单体验→配送体验→用餐体验→互动评论。** 每个触点都产生用户体验，任何一个环节的体验崩塌都会杀死复购。
27. **外卖的"产品"=菜品+包装+配送+服务。** 塑料袋包装→低档感，配送超时→愤怒感，客服态度差→被忽视感。这些体验直接决定"下次还来不来"。

### 【认知升维】

28. **万单店没有秘密，只有基本功做到极致。** 万单店的共性：出品标准化极高、线下硬实力强（出餐速度/品控/评价管理）、菜单精简聚焦、活动有节奏感、推广精准。不是某个"绝招"，而是每个环节都不拉垮。
29. **运营和产品的关系是循环驱动。** 数据驱动运营→运营驱动产品→产品驱动市场→市场反馈数据。运营告诉老板"该上什么产品、定什么价"，老板执行后看数据验证，再调整。这个循环永不停止。
30. **外卖的终极竞争力=成本效益+数据优势+存量复购。** 成本优势让你有定价空间，数据优势让平台愿意给你曝光，存量复购让你在固定用户池里越活越好。三者缺一不可，这是唯一可持续的盈利模型。
